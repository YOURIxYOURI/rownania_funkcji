1. Aby utworzyć połączenie z bazą danych musimy:
 - Dodać odwołąnie do biblioteki któa umożliwi nam połączenie sie z bazą danych np. Mysql.Data za pomocą słowa kluczowego using
 - tworzymy obiekt bedący MysqlConnection u ustawiamy własność Connection string w nastepujacy sposob "{server};{uid};{password};{database}"
 - otwieramy połączenie używając metody open() i tworzymy obiekt bedący komendą czyli MysqlCommand gdzie w konstruktorze zawieramy wczesniej przygotowany string z 
  kwerendą mysql i obiektem połączenia
 - wywołanie obiktu z gotową kwerendą możemy wykonać za pomocą metody ExecuteNonQuery/ExecuteQuery lub obiektu MysqlReader
 -zamykamy połączeni z baza używająć metody close na obiekcie połączenia
 
Przykład użycia:
Using Myslq.Data

MysqlConnection con = new();
con.ConnectionString = "server=localhsot;uid=root;pwd=;database=nazwa;";
con.Open()


String sql //kwerenda

MyslqCommand cmd = new(con, sql) 
Cmd.ExecuteNonQuery()
con.Close()
